📲 WHATSAPP INTELIGENTE + PERSONALIZAÇÃO AVANÇADA (OBRIGATÓRIO)

Criar um botão flutuante de WhatsApp totalmente personalizado e integrado à identidade visual do site.

🎨 BOTÃO FLUTUANTE:

- Cor de fundo: usar a cor principal do site (NÃO usar verde padrão do WhatsApp)
- Ícone: branco ou cor de alto contraste
- Sombra suave com a cor principal:
  box-shadow: 0 4px 20px var(--cor-sombra)

- Hover:
  - leve escurecimento OU glow
  - pequena escala (scale 1.05)

- Animação:
  - efeito de pulso leve e elegante
  - pode incluir micro “notificação” (bolinha animada)
